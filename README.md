WorkNest - Freelancing Platform

WorkNest is a modern freelancing platform built using React, TypeScript, Vite, TailwindCSS, and shadcn/ui. It allows clients to browse freelancers, send requests, and view profiles, while freelancers can manage their details and interact with clients. The project is designed for fast performance, clean UI, and smooth user experience with real-time data handling.

Features:
-> User authentication (Client & Freelancer)

-> Freelancer profile management

-> Client request system

-> Modern UI with shadcn/ui & TailwindCSS

-> Fast performance with Vite

-> Responsive and clean dashboard layouts
